API Surface

APIs are the primary product surface. Usage-metered, rights-aware, designed for automation.

Dataset Access API

Fine-grained control over usage, scope, and licensing enforcement.

const dataset = await harbor.datasets.get({
id: "audio-speech-v4"
});

RAG Query API

Query datasets contextually using multimodal retrieval pipelines.

const ctx = await harbor.rag.query({
query: "person opening door"
});

Annotation API

Submit media for annotation or retrieve existing annotations.

const job = await harbor.annotate({
media_url: "s3://..."
});

Streaming API

Real-time stream access for live media workflows.

const stream = harbor.stream({
source: "rtmp://..."
});
